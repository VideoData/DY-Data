# 抖音直播广场列表接口

##  请求API：

- `http://主机地址/douyin/webcast/feed?token=xxx&channel_id=221003`

##  请求方式：

- GET

##  参数：

| 参数名 | 必选 | 类型 | 说明 |
| :--- | :--- | :--- | --- |
| token | 是 | string | 接口授权码 |
| channel_id | 是 | int | 频道id |
| cursor | 否 | int | 翻页游标，根据结果返回的max_time传入作为下一页翻页参数，初始为0 |
