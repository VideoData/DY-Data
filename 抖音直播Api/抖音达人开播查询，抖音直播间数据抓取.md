# 抖音直播间状态查询，抖音直播间数据抓取

# 接口：直播开播查询

### 请求Api
```http
http://主机地址/douyin/liveroom/status?token=xxx&room_id=6843198199583378191,6872323400263797518
```

### 

### 请求方式
```http
GET
```

### 

### 参数
| 字段 | 类型 | 说明 |
| --- | --- | --- |
| token | string | 接口授权码 |
| room_id | int | 直播间id |

 

### 返回示例
```json

{
  "code": 200,
  "data": {
    "data": [
      {
        "alive": false,
        "room_id": 6843198199583378191
      },
      {
        "alive": true,
        "room_id": 6872323400263797518
      }
    ],
    "extra": {
      "now": 1600096174925
    },
    "status_code": 0
  },
  "msg": "success"
}
```
