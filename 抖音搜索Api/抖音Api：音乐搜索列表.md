## 抖音音乐搜索列表Api：

### 请求Api
```http
http://主机地址/douyin/search/music?token=xxx&keyword=热巴&cursor=20

```

### 

### 请求方式
```http
GET
```

### 

### 参数
| 字段 | 类型 | 说明 |
| --- | --- | --- |
| token | string | 接口授权码 |
| keyword | string | 搜索关键词 |
| cursor | int | 翻页游标，根据结果返回的cursor传入作为下一页翻页参数，初始为0 |
